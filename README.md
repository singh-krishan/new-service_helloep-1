# new-service_helloep-1

A Python microservice

Generated by IDP Platform

## Local Development

```bash
# Install dependencies
pip install -r requirements.txt

# Run the application
python -m uvicorn src.main:app --reload --port 8002
```

The application will be available at `http://localhost:8002`

## Endpoints

- `GET /` - Root endpoint
- `GET /health` - Health check endpoint

## Running Tests

```bash
pip install pytest pytest-asyncio httpx
pytest tests/ -v
```

## Docker

Build and run with Docker:

```bash
docker build -t new-service_helloep-1 .
docker run -p 8002:8002 new-service_helloep-1
```

## Deployment

This project is configured for automated deployment using:
- **GitHub Actions**: Builds and pushes Docker image on push to main
- **ArgoCD**: Automatically deploys to Kubernetes using Helm chart
- **Helm Chart**: Located in `helm/` directory

The CI/CD pipeline will:
1. Run tests
2. Build Docker image
3. Push to GitHub Container Registry
4. ArgoCD syncs and deploys to Kubernetes

## Monitoring

The service is configured with Prometheus metrics annotations for automatic monitoring integration.
